Recipe Manager

The Recipe Manager is a project that I made for myself and my girlfriend.
We wanted to start cooking together and naturally looked up a lot of recipes in the process.
I realized that trying to keep track of all of these recipes across all of these different sites was rather unreliable.
What if we lose the link? What if the recipe gets taken down for some reason?
I had the idea to create a site where we could just copy all of the recipes that we find into our own online repository and access them at will.
The primary technologies used are:

Front-End:
  • Vue.js as the JS framework
  • Bulma as the CSS framework
Back-End:
  • Node.js with Express for the back-end engine
  • Postgres as our database

While the site was made for my personal use, it's open for anyone to make an account and use.
